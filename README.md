# Python Programming Lab - Complete Guide

A comprehensive web-based learning platform for Python Programming Lab with **50+ programs**, **interactive code runner**, and **visualizations**.

## Features

- **14 Tasks** covering all Python lab topics
- **In-browser Python compiler** (Pyodide) - no server needed
- **Run** button to execute code directly in the browser
- **Visualize** button for visual output (patterns, charts, graphs)
- **Simple explanations** for every program
- **Dark/Light theme** support
- **Responsive design** - works on mobile
- **Search** functionality

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Navigate to any task and click **Run**

- HTML5, CSS3, JavaScript
- Pyodide (Python in WebAssembly)
